## Featured Projects

**🌟 [LoveMeter](https://github.com/SenthJeya/LoveMeter)**  
A beautiful, privacy-first Love Calculator built with React 19, Vite & Tailwind CSS. Fully local, dark-mode UI with confetti and animations.

**🌟 [University-Event-Managment](https://github.com/SenthJeya/University-Event-Managment)**  
Role-based web application with dashboards for Vice Chancellor, Dean, HOD, staff & students. Multi-level approval workflow (React • Node.js • Express • MongoDB • JWT).

**🌟 [Express-Foods](https://github.com/SenthJeya/Express-Foods)**  
Dynamic food ordering & management system with admin & user panels (PHP • MySQL • Bootstrap • HTML/CSS/JS).

**🌟 [DUTU](https://github.com/SenthJeya/DUTU)**  
Minimalist MERN task management app with JWT authentication, CRUD operations & clean UI.

**🌟 [Crown-Travels](https://github.com/SenthJeya/Crown-Travels)**  
React Native travel app for Sri Lankan tourist destinations, bookings & payments (Expo • Firebase • Stripe).
